# learn 4 vue

带你从零入门 vue.js

### 项目启动

```
git clone https://github.com/onaug6th/learn4vue

cd learn4vue

npm install 

npm run dev
```
默认会开启8080端口，访问localhost:8080即可打开项目。

### 构建生产资源

Run `npm run build` to build the project. The build artifacts will be stored in the `dist/` directory.
